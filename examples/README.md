# Примеры использования NotKinopoisk PHP Wrapper

Эта директория содержит примеры использования библиотеки NotKinopoisk PHP Wrapper для работы с Kinopoisk API.

## 📋 Список примеров

### 1. `basic_usage.php` - Базовое использование

**Описание**: Демонстрирует основные возможности библиотеки

- Получение информации о фильмах
- Работа с персоналом фильмов
- Поиск по ключевым словам
- Получение популярных фильмов
- Работа с фактами, видео и отзывами
- Получение изображений и связанных фильмов
- Информация об API ключе

**Запуск**: `php examples/basic_usage.php`

### 2. `advanced_search.php` - Продвинутый поиск

**Описание**: Демонстрирует возможности поиска и фильтрации

- Получение доступных фильтров
- Поиск с множественными фильтрами
- Работа с сериалами и фильмами
- Топ-250 фильмов
- Коллекции фильмов
- Пагинация результатов
- Статистика поиска

**Запуск**: `php examples/advanced_search.php`

### 3. `persons_and_media.php` - Работа с персонами и медиа

**Описание**: Демонстрирует работу с персонами и медиа-контентом

- Поиск и детальная информация о персонах
- Персонал фильмов (актеры, режиссеры, продюсеры)
- Изображения фильмов (постеры, кадры, фан-арты)
- Видео контент (трейлеры, виджеты)
- Факты и отзывы
- Внешние источники
- Статистика медиа-контента

**Запуск**: `php examples/persons_and_media.php`

### 4. `error_handling.php` - Обработка ошибок

**Описание**: Демонстрирует различные типы ошибок и их обработку

- Обработка несуществующих ресурсов
- Валидация параметров
- Обработка лимитов API
- Общая обработка исключений
- Рекомендации по обработке ошибок

**Запуск**: `php examples/error_handling.php`

### 5. `configuration_examples.php` - Конфигурация

**Описание**: Демонстрирует различные способы настройки клиента

- Передача API ключа напрямую
- Использование переменных окружения
- Работа с .env файлами
- Дополнительная конфигурация HTTP клиента
- Конфигурация для продакшена
- Логирование запросов
- Рекомендации по безопасности

**Запуск**: `php examples/configuration_examples.php`

### 6. `enums_usage.php` - Использование перечислений

**Описание**: Демонстрирует работу с перечислениями (Enums)

- Типы контента
- Типы изображений
- Порядок сортировки
- Типы коллекций
- Месяцы и другие перечисления

**Запуск**: `php examples/enums_usage.php`

### 7. `with_dotenv.php` - Работа с .env файлами

**Описание**: Демонстрирует интеграцию с vlucas/phpdotenv

- Загрузка переменных из .env файла
- Безопасное хранение API ключей
- Конфигурация для разных окружений

**Запуск**: `php examples/with_dotenv.php`

## 🚀 Быстрый старт

### Предварительные требования

1. **Установите зависимости**:

   ```bash
   composer install
   ```

2. **Получите API ключ**:

   - Зарегистрируйтесь на [https://kinopoiskapiunofficial.tech](https://kinopoiskapiunofficial.tech)
   - Получите бесплатный API ключ

3. **Настройте API ключ**:

   **Вариант 1: Переменная окружения**

   ```bash
   export KINOPOISK_API_KEY=your_api_key_here
   ```

   **Вариант 2: .env файл**

   ```bash
   echo "KINOPOISK_API_KEY=your_api_key_here" > .env
   ```

### Запуск примеров

```bash
# Базовое использование
php examples/basic_usage.php

# Продвинутый поиск
php examples/advanced_search.php

# Работа с персонами и медиа
php examples/persons_and_media.php

# Обработка ошибок
php examples/error_handling.php

# Конфигурация
php examples/configuration_examples.php
```

## 🔧 Настройка для разработки

### Рекомендуемая структура проекта

```
your-project/
├── .env                    # Переменные окружения
├── composer.json           # Зависимости
├── src/                    # Ваш код
├── examples/               # Примеры использования
└── .gitignore             # Исключения Git
```

### .env файл

```env
# API ключ для Kinopoisk API
KINOPOISK_API_KEY=your_api_key_here

# Дополнительные настройки
KINOPOISK_API_BASE_URL=https://kinopoiskapiunofficial.tech
```

### .gitignore

```gitignore
# Переменные окружения
.env
.env.local
.env.*.local

# Зависимости
vendor/

# Логи
*.log

# Кэш
.phpstan.cache
```

## 🛠️ Отладка

### Включение отладки

```php
// В начале скрипта
error_reporting(E_ALL);
ini_set('display_errors', 1);

// Логирование запросов
$config = [
    'on_stats' => function ($stats) {
        echo "Запрос: " . $stats->getEffectiveUri() . "\n";
        echo "Время: " . $stats->getTransferTime() . " сек\n";
    },
];
$client = new Client($apiKey, $config);
```

### Проверка API ключа

```php
try {
    $apiInfo = $client->users->getApiKeyInfo();
    echo "Тип аккаунта: " . $apiInfo->accountType->value . "\n";
    echo "Осталось запросов: " . $apiInfo->getRemainingTotalQuota() . "\n";
} catch (InvalidApiKeyException $e) {
    echo "Ошибка: Неверный API ключ\n";
}
```
